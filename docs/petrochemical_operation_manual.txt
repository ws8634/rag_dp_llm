# 石化生产基础信息
石化生产的主要原料包括原油、天然气、煤炭和生物质等。
原油经过蒸馏、裂化、加氢等工艺，可生产汽油、柴油、乙烯、丙烯等基础化工原料。
天然气主要用于生产合成氨、甲醇和乙烯，也是重要的清洁能源。

# 金陵石化核心装置工艺参数
金陵石化350万吨炼化装置核心工艺：
1. 原油裂化温度约450℃，反应压力5.0MPa；
2. 天然气合成氨核心反应温度450℃，催化剂为铁基催化剂；
3. 炼化装置的副产品包括丙烷、丁烷，年产能约50万吨。

# 天然气的主要用途
天然气的主要用途：
1. 民用燃料，用于居民做饭、取暖；
2. 工业原料，用于生产合成氨、甲醇等化工产品；
3. 发电燃料，用于燃气轮机发电，效率约55%；
4. 交通燃料，用于压缩天然气（CNG）汽车，污染排放低。

# 常减压蒸馏装置生产运行规程（工业级）
## 1. 装置基本信息
装置名称：1000万吨/年常减压蒸馏装置
设计单位：中国石油工程建设有限公司华东分公司
投产时间：2020年6月
核心功能：将原油分离为汽油、煤油、柴油、蜡油、渣油等馏分，为后续二次加工装置提供原料。

## 2. 核心工艺参数
### 初馏塔
- 塔顶温度：105~110℃
- 塔顶压力：0.05~0.08MPa（表压）
- 进料温度：220~230℃

### 常压塔
- 常压炉出口温度：360~370℃
- 侧线一（煤油）抽出温度：200~210℃
- 塔底温度：350~355℃

### 减压塔
- 减压炉出口温度：390~400℃
- 塔顶残压：3~5kPa（绝压）
- 真空度：≥99.3kPa

## 3. 常见故障及处理
### 常压塔塔顶压力升高
- 原因：塔顶冷却器结垢、回流量过大、进料量突增；
- 处理：清洗冷却器、降低回流量、稳定进料量。

### 减压塔真空度下降
- 原因：蒸汽喷射器故障、系统泄漏、塔底吹汽量过大；
- 处理：切换备用喷射器、检查密封点、降低吹汽量。
